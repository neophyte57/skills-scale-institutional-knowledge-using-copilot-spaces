# Change Management Checklist

- [ ] Change clearly described and scoped
- [ ] Stakeholders identified and notified
- [ ] Impact analysis performed
- [ ] Communication plan developed
- [ ] Training/enablement plan created (if needed)
- [ ] Risk mitigation strategies considered
- [ ] Change approved by accountable roles
- [ ] Implementation planned and scheduled
- [ ] Change communicated pre- and post-launch
- [ ] Lessons learned captured and documentation updated
